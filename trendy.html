Este desarrollo nace de la necesidad expandir la cartera de clientes al mercado online.

Este desarrollo cuenta con un "administrador" en el cual se podrán agregar, editar o eliminar partes del e-commerce tales como el carrusel, secciones principales, categorías, líneas, colores, productos, entre otras. También desde este administrador se podrán consultar las ventas, visitas al e-commerce, el inventario de productos, personas suscritas al boletín informativo, todo esto a través de reportes que se pueden exportar a Excel o a PDF, también se podrá revisar las compras hechas por el cliente, agregar un número de guía, atender tickets de soporte, regular reseñas hechas por clientes, denegar acceso a clientes, entre otros.

Entre las características de este e-commerce se puede destacar lo siguiente:
- Control de inventario
- Integración de pago BBVA y Mercado Pago
- Carrito de compras
- Lista de deseos para clientes registrados
