Este desarrollo nace de la necesidad de vender a clientes recurrentes con la mínima intervención humana posible, buscando al mismo tiempo automatizar cálculos, ampliar cartera de clientes, hacerse un espacio en el mercado de telas online y unir las funcionalidades de un software de punto de venta que maneja la empresa con las funcionalidades de un e-commerce.

Este desarrollo cuenta con un "administrador" en el cual se podrán agregar, editar o eliminar partes del e-commerce tales como el carrusel, secciones principales, categorías, líneas, colores, productos, entre otras. También desde este administrador se podrán consultar las ventas, visitas al e-commerce, el inventario de productos, personas suscritas al boletín informativo, todo esto a través de reportes que se pueden exportar a Excel o a PDF también se podrá revisar las compras hechas por el cliente, agregar un número de guía, atender tickets de soporte, regular reseñas hechas por clientes, crear vendedores, gestionar descuentos a clientes preferenciales, denegar acceso a clientes, entre otros.

Entre las características de este e-commerce se puede destacar lo siguiente:
- Sincronización de rollos con base de datos de software de punto de venta
- Venta por rollos
- Venta por metros (algoritmo que busca y selecciona los rollos cuya suma de metros sea más acercada a la agregada al carrito)
- Gestión de descuentos porcentuales a clientes preferentes
- Carrito de compras
- Lista de deseos para clientes registrados
